# Linux dotfiles


**Arandr**
```
~/.screenlayout
```

**i3**
```
~/.config/i3/config
```

**lightdm**
```
/etc/lightdm/lightdm.conf
```

**logind**
```
/etc/systemd/logind.conf
```
**Picom**
```
/etc/xdg/picom.conf
```

**Polybar**
```
~/.config/polybar/config
```

**Xinit*
```
/etc/X11/xinit/xinitrc
```
